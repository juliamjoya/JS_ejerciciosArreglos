Ejercicios Arreglos JavaScript

A continuación se describe cada uno de los ejercicios sobre condicionales en JavaScript y la solucion a cada uno se encuentra en el archivo llamado "ejerciciosCondicionales.js":

Ejercicios 1: Crear un array vacio, luego generar 10 números al azar y guardarlos en un array. Mostrar en consola el resultado del array.

Ejercicios 2: El usuario deberiá ingresar un string con varias palabras separadas por coma en un popup y se deben convertir en un array, (el usuario ingresa:'1,2,3,4,5' y se convierte en [1,2,3,4,5]). Mostrar en consola dicho resultado.

Ejercicios 3: De acuerdo al array [10,40,30,20,15,5] imprime lo siguiente: El arreglo ordenado de menor a mayor, muestra el numero menor y el numero mayor. Tip: Busca en google los metodos JavaScript que regresan el mayor y menor elemento de un arreglo.
